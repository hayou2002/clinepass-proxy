# ClinePass Proxy 更新日志

## v2.0 (2026-07-15)

### 新增功能
- 多Key轮询调度，支持动态增删
- 429限流自动切换Key，冷却后自动恢复
- Web管理面板（白色主题）
- 一键健康检测，区分周上限/普通限流/认证错误
- 模型信息展示（能力、上下文、最大输出等）
- 全局连接池优化

### 修复问题
- 修复非流式响应需要解包 ClinePass data 包装层
- 修复模型名需要 cline-pass/ 前缀
- 修复 retry 变量未定义错误
- 保留 reasoning 字段支持 CherryStudio 思考过程

### 技术改进
- 使用 aiohttp TCPConnector 复用连接
- 异步处理所有IO操作
- 完善错误处理和日志记录

## v1.0 (2026-07-05)

- 初始版本，Go 二进制实现
- 单Key支持
- 基础代理转发
